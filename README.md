# BC Remittance Limited (BCRemit)
BCRemit is an online money transfer service (via mobile app, web and call centre) that allows you to send money using your mobile phone, tablet or laptop/PC from the UK to the Philippines. Money can be sent via Credit-to-Bank Account and Cash Pick-up.
BCRemit is online only to provide you low cost, secure, fast and convenient service. With BCRemit you can do your money transfer anywhere and anytime you have access to the internet.
BCRemit is registered and regulated by the UK Financial Conduct Authority (FCA) and the HMRC.
## How to Run the Project
`npm install`
`npm start`
`npm run android`
`npm run ios`
